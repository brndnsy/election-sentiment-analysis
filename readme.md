# Sentiment Analysis using ML
An end-to-end data science project about 2019 UK General Election.  

## Visualisation

https://brndnsy.pythonanywhere.com


### Some Prerequisites

* Python 3.7 (or later)
* nltk
* dash
* pandas
* matplotlib
* plotly
* sklearn
* tqdm


#### Install libraries and run program

```
pip install -r requirements.txt
python3 main.py
```
